# AngularApp3

子コンポーネントの`cockpit`でデータ入力を受付
→ データを一度, 親コンポーネントの`app`に渡す
→ 子コンポーネントの`server-element`で渡って来たデータを描画する

- `cockpit`は入力受付.
- `server-element`は描画.
- `app`は全体まとめ.

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.3.8.
